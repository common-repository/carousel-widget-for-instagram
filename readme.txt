This plugin displays a list of recent Instagram pictures from multiple accounts and hashtags into one or multiple widgets.

== Description ==

*Carousel Widget For Instagram* allows you to easily display pictures from multiple Instagram accounts and hashtags sorted chronologically.

**Features**

*   Offers builtin themes to quickly customize how the widget looks
*   Displays feeds from as many different users and hashtags as you want, simultaneously, sorted by published date
*   Caches data for faster page loads and refreshes the cache on a custom timer
*   Lets you decide if you want to show/hide the widget title

 

== Installation ==

1.  Install the plugin through the WordPress plugins screen directly.
2.  Activate the plugin through the 'Plugins' screen in WordPress.
3.  Add the *Carousel Widget For Instagram* to an active sidebar via the *Appearance*->*Widget* screen.
4.  Configure the widget to fit your needs.
== Frequently Asked Questions ==

**Where do I put my Instagram API key?**

You don't need to create an Instagram app and provide an API key to use this plugin; it retrieves data from public HTTP requests.

**Where can I configure the settings for a widget?**

Each widget has its own settings and can be different from each other. Each widget has a settings form available from the
*Appearance->Widget* screen.
